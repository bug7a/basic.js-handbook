object.load(filePath: String)

The image file takes a while to load.
• When the loading is complete, the function defined with onLoad() is called.

object.autoSize
Automatically resizes the uploaded image.
Type: Integer Default: 1
ValueDescription
0Fit the specified size of the object.
1Original size.
2Half the original size.
3One-third of the original size.
......
The sizes of the object are calculated when it is loaded.
The autoSize value sets as 0 when the width or height of the image is changed.
On high resolution mobile devices; In order to obtain a better quality view, image files can be saved large and reduced when loaded onto the screen.

object.naturalWidth
Returns the original width of the image.
Type: Integer

• Must be used inside the onLoad() function.
If object.autoSize = 1, the .width and .naturalWidth properties will have the same value when the image is loaded.

object.naturalHeight
Returns the original height of the image.
Type: Integer

• Must be used inside the onLoad() function.

object.space

Type: Integer Default: 0 Min: 0 Max: 50

• Value is used as a percentage of image sizes.
• Uses padding property in CSS.
